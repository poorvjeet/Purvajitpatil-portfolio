# Purvajit Patil - Portfolio

A modern, visually stunning portfolio website built with React, featuring custom animations, Aurora background effects, and a dark theme aesthetic.

## 🛠️ Tech Stack

- **Frontend**: React 18, Vite
- **Styling**: CSS Variables, Custom CSS
- **Animations**: Three.js (Aurora), Framer Motion concepts
- **Build**: Vite

## 🚀 Features

- 🎨 Custom Aurora background effect
- 🖱️ Magnetic button interactions
- ✨ Text scramble & reveal animations
- 📱 Fully responsive design
- 🌙 Dark theme with teal/blue accents
- 💫 Tilt card effects

## 📦 Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
